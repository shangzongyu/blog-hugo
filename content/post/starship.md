# 介绍

# 安装


```sh

# Arch Linux
pacman -S starship

# FreeBSD
pkg install starship

# macOS
brew install starship
```

# 配置

```
# ~/.bashrc
eval "$(starship init bash)"

# ~/.tcshrc
eval `starship init tcsh`

# ~/.zshrc

```
